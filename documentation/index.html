Installation


Or follow bellow process

Once you download .zip file from CodeCanyon, open it and find ovoo.zip inside.
Copy ovoo.zip somewhere on your hard drive and then follow steps below:

To install using the built-in installer:

  1. Login to cPanel > File Manager.
  2. Under Upload tab, click on "Choose File" and choose ovoo.zip from your hard drive.
  3. Open Any Internet Browser to install the Ovoo.
  4. Type your web address and hit enter.Example(www.yourdomain.com)
  5. If this is the first time you've not installed ovoo, you will get installation page.
  6. Step-1:Check File Write Permission Then Click Next to continue with the installation.
  7. Step-2:Enter Purchase Information Then Click Next to continue next step.
  8. Step-3:Enter Database login credential information Then Click Next to continue next step.
  9. Step-4:Enter Admin login credential information Then Click Start Installation to Finish Installation.
  10. If successful, login To admin dashboard to manage your system. Retry if you are fail to maintain full steps.

To upload a Ovoon using FTP:

  1. Login to your hosting via your FTP client.
  2. Navigate to your public_html directory.
  3. Extract ovoo.zip and copy Ovoo's files to public_html directory on your host.
  4. Open Any Internet Browser to install the Ovoo.
  5. Type your web address and hit enter.Example(www.yourdomain.com)
  6. Follow steps to complete.

Installation Requirment

  • PHP 5.4 or Higher.
  • MySQLi.
  • GD Library.
  • Simple XML.
  • cURL.
  • mbstring
  • allow_url_fopen.

Setup Note

In order to work, OVOO must have at least:

  • 'application/config/config.php' is writable
  • 'application/config/database.php' is writable
  • '/uploads' directory is writable
  • CURL(php extension) must be enable

You will find more info right below on why to set up these...

IMDB Import

  1. Login to administrator dashboard.
  2. Login to www.themoviedb.org & find your movie.
  3. Then copy tmdbID from browser address bar.
  4. Goto Videos » New Video
    Note: Click on image to view more larger.
  5. Paste IMDb ID here & click import.

New Movie

  1. Login to administrator dashboard.
  2. Goto Videos » New Video
  3. Enter video information & click create.

New TV-Series

  1. Login to administrator dashboard.
  2. Goto TV-Series » New Video
  3. Enter video information & click create.

New Page

  1. Login to administrator dashboard.
  2. Goto Pages » New Page
  3. Enter page information & content ,then click create.

New Post

  1. Login to administrator dashboard.
  2. Goto Post » New Post
  3. Enter post information & content, then click create.

Compatibility & Requirements

Ovoo was made using PHP 5.6 and MySQL 5.7.11 , but since it uses common PHP functions, it should work on older versions as well, but there’s no guarantee.

Minimum System Requirements: 

PHP Version: 5.4+
                                                                        CURL ENABLED (php extension)
                                                                        Apache - rewrite_module (Apache Module) Must enable for SEO Friendly URL
                    Browser Support: Google Chrome | Mozilla Firefox | Opera | Safari | Internet Explorer 8+

Recommended: 

PHP Version: 5.6+
                                                                        CURL ENABLED (php extension)
                                                                        MySQL Version: 5.7.x
                                                                        Apache - rewrite_module (Apache Module) Must enable for SEO Friendly URL
                    Browser Support: Google Chrome | Mozilla Firefox | Opera | Safari | Internet Explorer 9+
